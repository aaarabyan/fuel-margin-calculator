# ⛽ Fuel Margin Calculator

Калькулятор маржинальности топлива для расчёта прибыли и анализа рентабельности.

## 🚀 Возможности

- 📊 Расчёт чистой прибыли и маржи на литр
- 🔄 Автоматическая конвертация тонны ↔ литры
- 📈 Графики чувствительности и структуры затрат
- 💾 Сохранение сценариев в браузере
- 📱 Адаптивный дизайн (PWA-ready)

## 💻 Технологии

- React 19 + TypeScript
- Vite 7
- Tailwind CSS 4
- Recharts

## 🛠️ Установка

```bash
npm install
npm run dev
```

## 📦 Сборка

```bash
npm run build
npm run preview
```
